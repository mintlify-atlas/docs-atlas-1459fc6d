---
title: "Get Comment by ID"
api: "GET /comments/:id"
---

Retrieve a specific comment by its ID, including any threaded replies.

## Authentication

This endpoint does not require authentication.

## Path Parameters

<ParamField path="id" type="string" required>
  The ID of the comment
</ParamField>

## Response

<ResponseField name="id" type="string">
  The unique identifier for the comment
</ResponseField>

<ResponseField name="body" type="string">
  The content of the comment
</ResponseField>

<ResponseField name="filePath" type="string">
  The file path where the comment was made
</ResponseField>

<ResponseField name="lineNumber" type="number">
  The line number in the file
</ResponseField>

<ResponseField name="pullRequestId" type="string">
  The ID of the pull request
</ResponseField>

<ResponseField name="userId" type="number">
  The ID of the user who created the comment
</ResponseField>

<ResponseField name="parentCommentId" type="string" optional>
  The ID of the parent comment if this is a reply
</ResponseField>

<ResponseField name="resolved" type="boolean">
  Whether this comment thread has been resolved
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 timestamp of when the comment was created
</ResponseField>

<ResponseField name="updatedAt" type="string">
  ISO 8601 timestamp of when the comment was last updated
</ResponseField>

<ResponseField name="user" type="object">
  The user who created the comment
  
  <ResponseField name="id" type="number">
    User ID
  </ResponseField>
  
  <ResponseField name="username" type="string">
    Username
  </ResponseField>
  
  <ResponseField name="name" type="string">
    Display name
  </ResponseField>
  
  <ResponseField name="avatarUrl" type="string">
    URL to the user's avatar image
  </ResponseField>
</ResponseField>

<ResponseField name="replies" type="array">
  Array of reply comments in the thread (same structure as parent comment)
</ResponseField>

## Example

```bash
curl https://api.diffy.com/comments/1
```
