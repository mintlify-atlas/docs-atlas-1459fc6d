---
title: "Update User"
api: "PATCH /users/{id}"
---

Update an existing user's information.

## Request

### Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the user to update
</ParamField>

### Headers

No authentication required.

### Body

All fields are optional. Only include the fields you want to update.

<ParamField body="githubId" type="string">
  GitHub ID of the user
</ParamField>

<ParamField body="email" type="string">
  Email address of the user (must be a valid email format)
</ParamField>

<ParamField body="username" type="string">
  GitHub username
</ParamField>

<ParamField body="name" type="string">
  Display name of the user
</ParamField>

<ParamField body="avatarUrl" type="string">
  URL to the user's avatar image
</ParamField>

<ParamField body="githubAccessToken" type="string">
  GitHub OAuth access token
</ParamField>

<ParamField body="githubRefreshToken" type="string">
  GitHub OAuth refresh token
</ParamField>

<ParamField body="installationId" type="integer">
  GitHub App installation ID
</ParamField>

## Response

Returns the updated user object.

<ResponseField name="id" type="integer" required>
  Unique identifier for the user
</ResponseField>

<ResponseField name="githubId" type="string" required>
  GitHub ID of the user
</ResponseField>

<ResponseField name="email" type="string">
  Email address of the user
</ResponseField>

<ResponseField name="username" type="string">
  GitHub username
</ResponseField>

<ResponseField name="name" type="string">
  Display name of the user
</ResponseField>

<ResponseField name="avatarUrl" type="string">
  URL to the user's avatar image
</ResponseField>

<ResponseField name="githubAccessToken" type="string" required>
  GitHub OAuth access token
</ResponseField>

<ResponseField name="githubRefreshToken" type="string" required>
  GitHub OAuth refresh token
</ResponseField>

<ResponseField name="installationId" type="integer">
  GitHub App installation ID
</ResponseField>

<ResponseField name="createdAt" type="string" required>
  ISO 8601 timestamp of when the user was created
</ResponseField>

<ResponseField name="updatedAt" type="string" required>
  ISO 8601 timestamp of when the user was last updated
</ResponseField>

## Example

```bash cURL
curl -X PATCH https://api.diffy.com/users/1 \
  -H "Content-Type: application/json" \
  -d '{
    "name": "John Smith",
    "installationId": 98765
  }'
```
