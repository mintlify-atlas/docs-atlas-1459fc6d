---
title: "Delete User"
api: "DELETE /users/{id}"
---

Delete a user from the system.

## Request

### Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the user to delete
</ParamField>

### Headers

No authentication required.

## Response

Returns the deleted user object.

<ResponseField name="id" type="integer" required>
  Unique identifier for the user
</ResponseField>

<ResponseField name="githubId" type="string" required>
  GitHub ID of the user
</ResponseField>

<ResponseField name="email" type="string">
  Email address of the user
</ResponseField>

<ResponseField name="username" type="string">
  GitHub username
</ResponseField>

<ResponseField name="name" type="string">
  Display name of the user
</ResponseField>

<ResponseField name="avatarUrl" type="string">
  URL to the user's avatar image
</ResponseField>

<ResponseField name="githubAccessToken" type="string" required>
  GitHub OAuth access token
</ResponseField>

<ResponseField name="githubRefreshToken" type="string" required>
  GitHub OAuth refresh token
</ResponseField>

<ResponseField name="installationId" type="integer">
  GitHub App installation ID
</ResponseField>

<ResponseField name="createdAt" type="string" required>
  ISO 8601 timestamp of when the user was created
</ResponseField>

<ResponseField name="updatedAt" type="string" required>
  ISO 8601 timestamp of when the user was last updated
</ResponseField>

## Example

```bash cURL
curl -X DELETE https://api.diffy.com/users/1
```
